Domain Names and ICANN

A domain name is a unique set of characters that identifies a specific website. Each domain name contains a top-level domain (TLD), like .com, .org and .net, and a subdomain of that top-level domain. For example, in the domain name njit.edu, .edu is the TLD and njit is the subdomain. Each domain name is associated with an IP address, which is a unique numerical label that identifies the location of the website.

Domain names and IP addresses are managed by an organization called the Internet Corporation for Assigned Names and Numbers (ICANN). ICANN manages the central repository for all IP addresses; they maintain and administer IP addresses for devices across the world. It is important to note that although ICANN oversees the use of Internet domains, it does not regulate the content and activities of the Internet.
